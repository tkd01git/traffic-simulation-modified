このリポジトリは、交通流シミュレーションの改良版とその解析ツール、結果データを含んでいます。以下に、各ファイルやスクリプトの使用方法と機能について説明します。

## 1. シミュレーションの使用方法

シミュレーションの基本的な使用方法やセットアップ手順については、元のリポジトリに含まれているREADMEファイルを参照してください。基本的な操作方法や依存関係のインストール手順が記載されています。



## 2. シミュレーションの改変

シミュレーションの道路構造やパラメータを変更するには、onramp.jsなどのJavaScriptファイルを編集します。これらのファイル内で、道路の形状や信号のタイミングなどを設定できます。具体的な編集方法については、各ファイル内のコメントや元のREADMEを参照してください。



## 3. データの取得と処理

シミュレーションの実行後、データはExcel形式で取得できます。以下の手順でデータを処理します。

1. データの収集: シミュレーションを実行し、生成されたデータを収集します。
2. Excelファイルへの変換: excelcompile.pyスクリプトを使用して、収集したデータをprodata.xlsxというExcelファイルにまとめます。

   excelcompile.py

   このスクリプトは、シミュレーションで取得したデータファイルを読み込み、prodata.xlsxとして保存します。



## 4. 結果の解析

生成されたprodata.xlsxを基に、さらなる解析を行います。

1. グラフフーリエ係数とF2の計算: prodatatoresult.pyスクリプトを使用して、各時刻のグラフフーリエ係数と、その中でF2の大きさの順位を計算し、result.xlsxというファイルに保存します。

   python ..prodatatoresult.py

   このスクリプトは、prodata.xlsxを入力として解析を行い、結果をresult.xlsxに出力します。



## 5. 結果の集計

特定の条件下での事象の数を集計するために、graphfouriersum.pyスクリプトを使用します。

1. 予兆の定義パラメータmの変更と集計: graphfouriersum.pyを実行し、予兆の定義におけるパラメータmを変更して、以下の4つの事象の数を各mについて計算します。

   - 渋滞かつ予兆あり
   - 渋滞かつ予兆なし
   - 非渋滞かつ予兆あり
   - 非渋滞かつ予兆なし

   python ..graphfouriersum.py

   このスクリプトは、result.xlsxを基に集計を行い、結果を表示または保存します。



## 6. 同梱されている結果ファイル

このリポジトリには、予備実験と本実験を合わせた計100回のシミュレーション結果が含まれています。各実験ごとにprodata.xlsxとresult.xlsxが生成されており、合計で200の結果ファイルが同梱されています。これらのファイルは、解析や検証の際に活用してください。
